# BIPA AI Lab 운영 상황판 키트

## 설치
1. 현재 GitHub Pages 저장소 루트에 `ops/` 폴더를 복사합니다.
2. `https://mecagun8.github.io/bipa-ai-lab/ops/` 로 접속합니다.
3. 자동 점검을 쓰려면 `.github/workflows/healthcheck.yml`도 복사하고 GitHub Actions 권한을 켭니다.

## 수정할 파일
- `ops/services.json`: 9개 솔루션의 담당자, 데이터 갱신 주기, 다음 행동 수정
- `ops/index.html`: 화면 문구·스타일 수정
- `ops/status.json`: Actions가 자동 갱신
