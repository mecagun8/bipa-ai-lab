💡분산 AI 서버란 무엇인가

AI 모델을 돌리려면 GPU(그래픽 처리장치)가 필요합니다. 보통은 고성능 GPU가 탑재된 비싼 서버 한 대를 사는 방식을 생각하지만, 작은 GPU 여러 대를 네트워크로 연결하면 같은 효과를 훨씬 저렴하게 낼 수 있습니다.

🏢

기존 방식 — 단일 고성능 서버

AI 전용 서버 1대 구매. GPU 1장이 모든 요청을 처리. 비싸고, 고장 시 전체 중단.

🌐

분산 방식 — 소형 PC 다수 연결

일반 PC에 게이밍 GPU 탑재 후 LAN으로 묶음. 요청을 자동으로 나눠 처리. 저렴하고 유연함.

쉬운 비유: AI 서버 1대를 사는 건 대형 트럭 1대, 분산 구성은 일반 승용차 10대입니다. 총 운반 능력은 비슷하지만 비용과 유연성이 다릅니다. 한 대가 고장나도 나머지 9대가 계속 운행합니다.

활용 범위

🎯이 구성으로 할 수 있는 것 / 없는 것

소형 GPU(RTX 3060, 12GB VRAM) 기반 분산 구성은 7B~14B 규모의 경량 모델을 운영합니다. 대형 모델(70B 이상)보다 처리 능력이 제한되므로 업무 유형에 따라 기대 수준을 맞춰야 합니다.

업무 유형적합도비고
공문·보고서 초안 작성, 요약✅ 적합7B 모델의 핵심 강점
번역 (한↔영 등 기본 번역)✅ 적합짧은 분량 기준
질의응답, 내부 규정 검색(RAG)✅ 적합문서 기반 검색과 병행 시 효과적
간단한 데이터 정리, 표 변환✅ 적합단순 형식 변환 수준
복잡한 법령 해석, 다단계 논리 추론⚠️ 제한적오류 가능성 있음, 검토 필요
코딩 보조, 코드 생성⚠️ 제한적간단한 스크립트 수준만 가능
장문 분석 (50페이지 이상 문서 전체)❌ 부적합컨텍스트 길이 한계
고품질 이미지·영상 생성❌ 부적합언어 모델 전용 구성
요약: 이 구성은 일상적인 문서 작업·요약·간단한 질의응답에 최적화되어 있습니다. ChatGPT 유료 플랜 수준의 고급 추론이나 복잡한 코딩 보조가 필요하다면 상위 모델 또는 클라우드 API 병행 검토가 필요합니다.

소프트웨어 구성

🧩사용하는 소프트웨어 (모두 무료·오픈소스)

1

Ollama — AI 모델 실행 엔진

각 PC에 설치하는 AI 모델 구동 소프트웨어. 명령어 한 줄로 모델 다운로드 및 실행. Windows 지원. 무료 오픈소스

2

Nginx — 요청 분배기 (로드밸런서)

모든 사용자 요청을 받아 10대의 PC 중 가장 여유로운 PC로 자동으로 전달. 설정 파일 몇 줄로 동작. 무료 오픈소스

3

Open WebUI — 직원 접속 화면

ChatGPT와 동일한 UX의 웹 채팅 인터페이스. 로그인, 대화 기록, 부서별 모델 설정 등 지원. 무료 오픈소스

4

AI 모델 — EXAONE / Gemma 등 한국어 지원 모델

LG AI 연구원의 EXAONE(한국어 특화) 또는 Google의 Gemma 등 12GB VRAM에서 실행 가능한 7B~14B 모델 사용. 무료 라이선스

구축 절차

🛠️구축 단계 (개략)

1

PC 수배 및 GPU 설치

RTX 3060 12GB 탑재 PC 10대 확보. 기존 교육용 PC를 야간·주말 활용하거나, 전용 PC 별도 수배. 각 PC는 내부 LAN으로 연결.

2

각 PC에 Ollama + 모델 설치

Ollama 설치 후 AI 모델 다운로드. PowerShell 스크립트로 10대에 일괄 배포 가능. 소요 시간: PC당 약 30분.

3

Nginx 로드밸런서 설정

중앙 PC(또는 서버) 1대에 Nginx 설치 후 10대 PC 주소를 등록. least_conn 알고리즘으로 자동 분배 설정.

4

Open WebUI 배포 및 직원 안내

웹 인터페이스 배포 후 내부 URL 공유. 직원은 별도 설치 없이 브라우저로 바로 사용 가능. Cloudflare 터널로 재택근무 접속도 지원.

자주 묻는 질문

❓예상 질문 및 답변

Q

"PC 한 대가 꺼지면 어떻게 되나요?"

Nginx가 자동으로 해당 PC를 제외하고 나머지 PC로만 요청을 분배합니다. 관리자 개입 없이 수 초 내 자동 전환됩니다.

Q

"정보 유출 위험은 없나요?"

모든 데이터가 기관 내부 서버에서만 처리됩니다. ChatGPT 등 외부 서비스와 달리 입력 내용이 외부로 전송되지 않습니다. 인터넷 차단 환경에서도 동작합니다.

Q

"기존 업무용 PC를 활용할 수 있나요?"

GPU 사양만 충족하면 가능합니다. 교육생이 사용하지 않는 야간·주말 시간대에만 AI 서버로 활용하는 방식도 검토할 수 있습니다.

Q

"몇 명이 동시에 사용할 수 있나요?"

PC 10대 기준, 10명이 동시에 별도 처리됩니다. 한 사람당 한 대가 전담하는 구조이므로 응답 속도가 일정하게 유지됩니다.

Q

"ChatGPT처럼 뭐든 다 되나요?"

아닙니다. 문서 요약·초안 작성·번역·간단한 질의응답에는 충분히 활용 가능합니다. 다만 복잡한 논리 추론이나 전문 코딩 보조는 경량 모델의 한계상 제한적입니다. 용도에 맞게 기대 수준을 설정하는 것이 중요합니다.
