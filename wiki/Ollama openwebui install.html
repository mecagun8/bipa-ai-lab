초기 설치 가이드

⚙️ 인프라 · 네트워크 ⏱ 소요시간 30분 📅 2026.05

개요

이 가이드는 Windows 10 + RTX 2070 Super 환경에서 로컬 LLM 스택을 구성하는 방법이에요. Ollama로 모델을 실행하고, Open WebUI로 ChatGPT 같은 인터페이스를 제공하는 구조예요.

💡
환경 기준
OS: Windows 10/11 · GPU: 상용 그래픽 카드 (VRAM 8GB 이상) · RAM: 32GB 이상 권장
VRAM 8GB는 사실상 최저 사양이에요. 7B 모델은 무난하지만 14B 이상은 버거울 수 있어요. VRAM이 클수록 더 큰 모델을 쾌적하게 쓸 수 있어요.

1단계 · Ollama 설치

Ollama 공식 사이트에서 Windows용 설치파일을 받아 설치해요.

1
설치파일 다운로드 및 실행
ollama.com → Download for Windows → 설치 실행
설치 완료 후 트레이 아이콘에 Ollama가 나타나요.
2
설치 확인
PowerShell에서 아래 명령어로 정상 설치 확인
PowerShell
ollama --version
ollama version is 0.x.x

2단계 · 모델 다운로드

VRAM 8GB (최저사양) 기준 추천 모델이에요. VRAM이 클수록 더 큰 모델을 쓸 수 있어요.

모델크기특징
gemma3:12b~7GB팀 메인 사용 모델, 한글 우수 · 8GB VRAM 한계치
qwen2.5:7b~4GB한글 성능 좋음, 여유 있게 동작
llama3.1:8b~5GB영문 범용
PowerShell
# 팀 기본 모델 설치 (시간 걸림 — 약 7GB)
ollama pull gemma3:12b

# 설치된 모델 목록 확인
ollama list

3단계 · Docker Desktop 설치 및 설정

1
Docker Desktop 설치
docker.com/products/docker-desktop → Windows용 다운로드 후 설치
설치 중 WSL2 백엔드 사용 옵션 체크 확인
2
NVIDIA Container Toolkit 활성화
Docker Desktop → Settings → Resources → GPU 활성화 체크
3
설치 확인
PowerShell
docker --version
Docker version 2x.x.x

4단계 · Open WebUI 설치 및 실행

Docker로 Open WebUI를 실행해요. 처음 실행 시 이미지를 다운로드하므로 시간이 걸려요.

PowerShell
docker run -d `
  -p 3000:8080 `
  --add-host=host.docker.internal:host-gateway `
  -v open-webui:/app/backend/data `
  --name open-webui `
  --restart always `
  ghcr.io/open-webui/open-webui:main
💡
--restart always 옵션
PC 재부팅해도 Docker Desktop 실행되면 Open WebUI가 자동으로 같이 켜져요.

5단계 · 접속 및 초기 설정

1
브라우저에서 접속
http://localhost:3000 접속 → 관리자 계정 생성 (처음 가입한 계정이 자동으로 관리자)
2
Ollama 연결 확인
Settings → Connections → Ollama API URL이 http://host.docker.internal:11434 로 되어 있으면 정상
3
모델 선택 후 채팅 시작
상단 드롭다운에서 gemma3:12b 선택 → 채팅 시작
✅
설치 완료!
채팅창에서 모델이 응답하면 정상이에요. 처음 응답은 모델 로딩 때문에 10~20초 걸릴 수 있어요.

자주 하는 실수 · 트러블슈팅

✗
Open WebUI에서 모델이 안 보임
Ollama가 실행 중인지 확인. 트레이 아이콘 확인 또는 ollama list 실행해보기
✗
localhost:3000 접속 안 됨
Docker Desktop이 실행 중인지 확인. docker ps로 open-webui 컨테이너 상태 확인
✗
GPU 안 씀 (응답 매우 느림)
Docker Desktop Settings → Resources → GPU 체크 여부 확인. NVIDIA 드라이버 최신 버전인지 확인
